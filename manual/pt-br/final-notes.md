# 25 — Notas Finais

O XForge.Scheduling fornece abstrações simples para agendamento de jobs. A implementação InMemory é ideal para desenvolvimento e testes.

## Ecossistema XForge

| Pacote | Propósito |
|--------|-----------|
| XForge.MediatR | Mediator in-process |
| XForge.Persistence | Repository, Unit of Work |
| XForge.Scheduling | Job scheduling |
| XForge.Security | Security primitives |
| XForge.State | Reactive state |
